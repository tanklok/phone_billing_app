# Информационная система для учёта междугородних телефонных переговоров

Клиент-серверное desktop-приложение для автоматизации учёта междугородних звонков, расчёта стоимости, применения скидок и управления доступом пользователей на основе ролей.

---

## О проекте

Разработан в рамках курсового проекта по дисциплине «Базы данных».  
**Технологический стек:**

- **Язык:** C++17
- **GUI:** Qt 6 (Widgets)
- **СУБД:** PostgreSQL 15+
- **Сборка:** CMake 3.16+
- **Система контроля версий:** Git

**Основные возможности:**

- Авторизация и регистрация пользователей с проверкой надёжности пароля.
- Ролевая модель (администратор, оператор, менеджер).
- Автоматический расчёт стоимости звонков на основе тарифов и скидок.
- Фильтрация и просмотр детализированной информации о звонках.
- Добавление и удаление звонков.
- Управление пользователями (только для администратора).

---

## Требования

### Для запуска готового приложения (без сборки)

- Windows 10/11 (или Linux, но сборка под Windows)
- Установленный PostgreSQL 15+ (сервер может быть удалённым)
- Клиентские библиотеки PostgreSQL (`libpq.dll` и др.) – входят в дистрибутив приложения

### Для разработки и сборки

- Qt 6.5+ (с компонентами Core, Widgets, Sql)
- CMake 3.16+
- Компилятор с поддержкой C++17 (MinGW или MSVC)
- Git (для клонирования репозитория)
- PostgreSQL 15+ (сервер и клиентские библиотеки)

## Установка и запуск

### 1. Скачайте готовый дистрибутив

Скачайте последний релиз из раздела [Releases](https://github.com/tanklok/phone_billing_app/releases) или соберите из исходников (см. раздел «Сборка»).  
Распакуйте архив в любую папку на компьютере.

### 2. Настройка базы данных PostgreSQL

Убедитесь, что сервер PostgreSQL запущен и доступен.  
Создайте базу данных и выполните SQL-скрипт инициализации.

Подключитесь к PostgreSQL через `psql` или pgAdmin и выполните:

```sql
CREATE DATABASE phone_billing;
\c phone_billing;
```
Затем выполните скрипт init_db.sql, который находится в папке с приложением или в репозитории (sql/init_db.sql). Он создаст все таблицы, индексы, роли и администратора.

Скрипт также создаёт учётную запись администратора:

- Логин: admin
- Пароль: admin (рекомендуется сменить после первого входа)

### 3. Настройка подключения к БД

В папке с приложением найдите файл config.ini.example и переименуйте его в config.ini.
Отредактируйте параметры подключения в соответствии с вашим сервером PostgreSQL:

```ini
[Database]
Host=localhost
Port=5432
Name=phone_billing
User=postgres
Password=ваш_пароль
```

Если вы используете удалённый сервер, укажите его IP-адрес вместо localhost.

### 4. Запуск приложения

Дважды щёлкните по PhoneBillingApp.exe. Откроется окно входа.
Введите логин admin и пароль admin для первого входа.

После успешной авторизации откроется главное окно, где вы сможете просматривать звонки, добавлять новые и управлять пользователями (только администратор).
